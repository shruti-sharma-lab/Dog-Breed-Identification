# Dog-Breed-Identification



##  Project Overview

This project is a **Deep Learning-based image classification system** that identifies the breed of a dog from an input image.

The model is trained on a dataset of dog images and uses computer vision techniques to accurately classify different dog breeds.

---

 Model Files: https://drive.google.com/drive/folders/1I8uPXsMIBwTfW_T9eySiVVfuJwINC5FS

---

##  Features

*  Predicts dog breed from uploaded image
*  Uses deep learning (CNN) for classification
*  High accuracy on trained dataset
*  Can be integrated with a web interface

---

##  Tech Stack

* **Programming Language:** Python
* **Deep Learning:** TensorFlow / Keras
* **Libraries:** NumPy, Pandas, Matplotlib
* **Model Type:** Convolutional Neural Network (CNN)

---



##  Installation & Setup

###  Step 1: Clone Repository


git clone https://github.com/shruti-sharma-lab/dog-breed-identification.git
cd dog-breed-identification


###  Step 2: Install Dependencies


pip install -r requirements.txt


###  Step 3: Run the Model


python app.py


---

##  Model Details

* Convolutional Neural Network (CNN)
* Image preprocessing (resizing, normalization)
* Trained on labeled dog breed dataset

---

##  Dataset

The dataset consists of multiple dog breed images used for training and testing the model.

---

##  Limitations

* Accuracy depends on dataset quality
* May misclassify similar-looking breeds
